Propriedade dos itens

Principais propriedades para o itens do flex.

Largura e altura para os itens com flex basis

O flex-basis altera largura e altura do elemento, usamos os mesmos inputs do width e height. Bloqueia o uso de dimensões específicas para os elementos internos. Usa a direção do eixo principal, ou seja, poderá ser altura ou largura dependendo da direção.

Crescimento e adaptação do item dentro do container

Crescimento do item dentro dos espaços vazios dentro do container. Podemos fracionar o espaço restante do container e distribuir da maneira que desejarmos entre os elementos.

Encolhimento e encaixe dos elementos

Usamos a propriedade flex-shrink para encolher os itens dentro do container. É usado somente quando há necessiade de encolher os itens, caso a distribuição tamanho dos itens e container esteja ok, não haverá encolhimento.

Shorthand flex

Podemos usar um shorthand flex para unir o flex-grow, flex-shrink e/ou flex-basis.

Alterando o tamanho de diversos elementos

Aditando o exemplo da div page. O basis é mais forte que as adaptações até o limite do tamanho do container.

Ordenando itens

Todos os elementos iniciam com order 0, mas podemos aumentar ou diminir (aceita negativos). Funciona somente de maneira visual e não estrutural, não altera o HTML gerando problemas de acessibilidade.

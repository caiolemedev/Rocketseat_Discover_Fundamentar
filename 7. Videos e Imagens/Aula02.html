Imagens no HTML

Imagens básico

Utilizar a tag img para inserir imagens indicando o atributo src (source), alt (texto alternativo caso não seja possível exibir a imagem), title (título que aparecerá quando descansar o mouse sobre a imagem), width ou height (optar por 1 para não deformar a imagem). Pode ser adicionada a um link para servir de âncora.

Intro sobre SVG

Foco dela é fazer imagens e trabalha com imagens vetorizadas (via algoritimo) e não rasterizadas (via pixels). Vetorizada é mais leve, mais detalhada, maior acessibilidade e SEO e pode ser editada via CSS. Mas é mais complicada para se trabalhar e com imagens mais complexas não é boa (ex: fotografias). Tendo o SVG diretamente na página reduz as chamadas externas e permite edição do elemento via CSS como por exemplo colocar fundo. Pode ser adicionada via tag img com arquivo externo.

Vídeos no HTML

Vídeos locais

Usar várias sources pois browsers diferentes aceitam diferentes tipos de video, colocar na ordem e especificar o tipo. Usar tags de controls, autoplay, width, height, loop, muted e poster.

Vídeos do Youtube ou outras fontes

Usar outra fonte como Youtube é bom pois economiza banda do seu servidor.

Audios

Funcionam de forma muito similar a tag vídeo com aceitando praticamente os mesmos parâmetros. Pode ser substituída por fontes externas também.

Usando a tag iframe

Permite colocar um conteúdo externo dentro do site, seja qual for. Alguns sites não autorizam este tipo de link. O compartilhar do youtube já permite compartilhamento dessa forma. A tag allow permite trazer configurações da fonte original. É possível colocar mapas do google via iframe também.

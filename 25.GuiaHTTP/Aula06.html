Headers

O que são headers?

São informações adicionais que possuem o formato campo:valor. Podemos ter diversas propriedades e combinações.

Entendendo pelo contexto

Dividimos em General, Response e Request. A maioria dos frameworks já possuem headers pré-deifinidos.

General

São headers usados tanto para o request quanto para o response. Mais comuns são: reques URL, request method, status code, remote address e referrer policy.

Request

Diversos, tais como: authority, method, path, accept, cookies, user-agent, etc.

Response

Temos outro conjunto de headers, por exemplo: cache-control, content-length, content-type, date, content-encoding, set-cookie, etc.

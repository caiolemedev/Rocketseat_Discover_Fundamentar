Mensagens

Foi definida no HTTP/1.1, legível e em texto e depois no HTTP/2 tem uma estrutura binária que otimiza a transmissão de dados.

Request

É o pedido feito pelo criente, precisamos na request line do method (verbo que executará o pedido), da versão do protocolo e da URI. As vezes são enviados headers e bodies também. O browser completa algumas informações como método e a versão do protocolo. Podemos manipular todos os dados do pedido via aplicação, browser, etc.

Response

Recebemos de volta a versão do protocolo, o status code, os headers e o status message. Todas as informações são preenchidas e enviadas pelo servidor. O servidor envia um texto HTML puro para ser "montado" pelo cliente.

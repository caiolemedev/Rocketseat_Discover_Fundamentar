Introdução

DOM (Document Object Model) é o HTML convertido para um objeto JS. API é o que representa e interage com o HTML. É a estrutura de dados do tipo árvore criada pelo navegador. Possui propriedades e métodos no JS, como qualquer outro objeto. O JS utiliza a DOM para se comunicar com o HTML e pode manipula-lo.

Estrutura de dados do tipo árvore, com nós interligando os itens e com pais e filhos. Exemplo: tag html tem dois filhos head e body e cada um com suas próprias tags filhos. Para acessar as propriedades de cada tag no JS usamos o ponto como qualquer outro objeto.

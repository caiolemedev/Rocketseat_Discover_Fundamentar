Eventos

Adicionando eventos via HTML

Usando a interação do usuário para disparar eventos. Por exemplo: onclick, ondbclick e mouseover.

Eventos de teclado

Adicionando eventos de teclado usando input.

Adicionando eventos via JS

Usando o próprio JS para disparar um evento via propriedade addEventListener e informando o evento e a função a ser executada.

Adicionando eventos via JS (Outros método)

Podemos adicionar diretamente uma função na propriedade onclick do elemento, mas só podemos fazer 1x. Caso tenha mais de 1 no código somente o último irá funcionar, exemplo no script.

Argumento event

Podemos capturar o tipo de evento, qual a tecla digitada (no caso do teclado), qual o target (quem dispara o evento). Podemos capturar o valor do input via propriedades do event.

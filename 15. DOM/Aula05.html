Navegando pelos elementos

parentElement e parentNode

Podemos selecionar o elemento pai do elemento através destes dois comandos.

Pegando elementos filhos

Usamos o childNodes para mostrar a lista de nós filhos. Os texts são os espaços que colocamos no código. Já o children traz somente os elementos filhos sem os textos, mas em formato HTMLCollection. Podemos selecionar os primeiros elementos via firstChild (mostra o espaço) ou o firstElementChild (mostra o primeiro elemento real), e de forma similar para o last.

Buscando irmãos

Usamos os comandos nextElementSibing e previousElementSibling para mostrar o próximo e o anterior respectivamente.

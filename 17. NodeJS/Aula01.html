O que podemos fazer com o NodeJS?

O NodeJS permite o desenvolvimento back e front end com JS, microserviços (aplicações completas), API (disponibilização de dados para o frontend), scripts, automação e até machining learning. Mas não é recomendado para grandes processamentos de CPU, exemplo: vídeos e fotos.

Vantagens

Muito rápido na execução e prototipagem, também tem uma excelente escalabilidade, usado no desenvolvimento de aplicações de ponta, usa o JS que é aplicado em vários lugares, Grade ecosistema e suporte. Temos casos famosos de uso: Netflix, Paypall, etc.

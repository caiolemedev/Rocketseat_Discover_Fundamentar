Atributos Globais

São atributos aplicados a todas as tags. Exemplo: class (classifica a tag), contenteditable (permite edição do texto), data-* (pode ser data-qlqr-coisa, usado no JS), hidden (deixa a tag escondida), id (1 por página), style (aplica CSS no próprio arquivo), tabindex (orienta o fluxo do tab) e title (define título para a tag, aparece com o mouse).

Aninhamento de Tags

As tags internas são "filhas" das tags externas, chamadas de "pais".

As tags são lidas na ordem que aparecem no código e existem tags que quebram linhas e outras que mantem o texto na mesma linha.

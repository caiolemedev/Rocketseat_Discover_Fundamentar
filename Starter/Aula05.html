Tag meta

Atributo charset: Define o conjunto de caracteres que será usado na página. Usar sempre UTF-8.

Atributos name e content: usado para portabilidade para dispositivos móveis.

Favicon

Abreviação para favorite icon e é um link para uma imagen específica. Para iphone e outros casos específicos utiliza-se outros tipos de atributo rel e tamanhos.

Meta para SEO

Otimização do conteúdo do site para motores de busca. Atibutos: name com conteúdo author, description com conteúdo de texto breve indicando o conteúdo da página, robots com conteúdo para direcionar os robos de busca (seguir ou não os links).

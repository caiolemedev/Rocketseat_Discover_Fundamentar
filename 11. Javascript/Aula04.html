Variáveis

Conhecendo as veriáveis

São nomes simbólicos utilizados para receber algum valor. Podemos criar de 3 modos diferentes: var, let e const (esse ultimo não permite a alteração de valor).

Tipos dinâmicos

JS é uma linguagem fracamente tipada e dinâmica. Variáveis não precisam ter um tipo previamente definido e os tipos podem ser mudados (dinâmica).

Scope e var

O escopo determina a visibilidade de uma variável no JS. Precisamos criar um block statement que também cria um novo escopo block-scoped. O var é global e pode funcionar fora do escopo de um bloco, o JS cria a var antes de qualquer coisa (hoisting). Atualmente evitamos usar var por conta desse fator.

Scope let e const

Let e const existem somente no bloco em que foram criadas, não sendo levadas para o escopo global. Caso o código esteja antes ele sabe que existe mas não pode utiliza-lo. Podemos atualizar o valor da variável let em escopos abaixo (não importa quantos) e ela passará a ter o novo valor para os demais escopos também. Podem existir 2 const com mesmo nome em escopos diferentes com valores diferentes.

Nomeando variáveis

O JS é case-sensitive e aceita cadeia de caracteres Unicode (caracteres especiais e acentos). Posso: iniciar com caracteres especiais, letras, colocar acentos, diferenciar com letras maiúsculas e minúsculas. Não posso: iniciar com números e colocar espaço vazio no nome. Ideal: Criar nomes com sentido, usar camelCase ou snake_case e preferencialmente utilizar o inglês.

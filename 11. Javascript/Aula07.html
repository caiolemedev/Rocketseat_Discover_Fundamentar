Manipulando dados

Prototype

Significa algo que foi criado antes, são algumas funcionalidades que já existem e ficam nessa cadeia de protótipo. Todo objeto criado referencia um objeto protótipo criado antes (padrão), ascendente. Podemos ter uma cadeia de protótipos (prototype chain) com diversos objetos agregando funcionalidades dos objetos anteriores. Para números precisamos tomar cuidado com o ponto para acessar as propriedades de proto.

Type conversion coersion

Type conversion (typecasting) explicitamente altera um tipo de dado para outro. Type coersion implicitamente o JS altera o tipo de dado. Alteram o tipo de dado, por exemplo string para number.

Strings e números

Transformando strings em números e vice versa. Usamos os comandos number() e string().

Contando caracteres e dígitos

Numeros não tem a propriedade length, para contar o número de dígitos precisamos primeiro converte-lo em string e depois acessar a propriedade.

Casas decimais

Podemos tirar as casas decimais via propriedade toFixed() mas isso transforma o número numa string, podemos converter de volta via Number(). No entanto se trocarmos o ponto pela vírgula e tentarmos converter novamente para número o JS não conseguirá e resultará em um NaN.

Maiúsculas e minúsculas

Podemos transformar toda a frase em maiúsculas ou minúsculas via comandos toUpperCase() e toLowerCase() respectivamente.

Encontrando palavras em frases

Usando a propriedade includes e passando o argumento que desejamos procurar o próprio JS resultará uma boolean indicando se existe ou não. É uma propriedade case sensitive, é recomendável converter o texto para todas maiúsculas ou minúsculas antes.

Separando strings

Transformar um texto em um array e depois de volta em texto mas com _ no lugar dos espaços. Usar a propriedade split para a primeira parte e o método join com o conector que você desejar.

Criando array com construtor

Usar o contrutor new Array com a quantidade de espaços desejada ou já inserir os valores também funciona.

Elementos do Array

Contar elementos do Array via length. Podemos também acessar os elementos do array e caso sejam objetos ou funções podemos acessar suas propriedades ou as executar.

Strings para arrays

Usando o método Array.from() para transformar uma cadeia de caracteres em elementos de um array, cada caracter ocupará um espaço.

Manipulando arrays

Adicionando e removendo itens no array via diferentes métodos em diferentes posições via push, unshift, pop, shift. Além de identificar a posição de um elemento do array e permitir "cortar" o array e remover elementos do meio do mesmo.

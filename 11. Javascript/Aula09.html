Condicionais e controle de fluxo

Controle de fluxo de aplicação

O programa é executado linha alinha de cima para baixo, mas algumas opções nos permitem manipular o fluxo de execução. Também podemos manipular os erros.

If e Else

If = se e Else = se não, são condições para execução de um bloco de código. Dica é transformar as condições em variáveis com significado. Os loops aceitam valores truthy e falsy.

Switch

Semelhante ao if e else mas usamos casos e break para finalizar o código e uma saída default para parâmetros que não estejam dentro dos casos.

Throw e Try/Catch

Podemos disparar um erro e tentar capturar erros na aplicação. O uso do throw, try e catch permite capturarmos os erros e seguir a operação da aplicação. Sem eles o erro trava todo o programa.

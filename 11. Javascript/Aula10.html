Estruturas de repetição

For

Estrutura de repetição for, para execula-la precisamos de uma inicialização, uma finalização e um incremento a ser feito para cada looping. O break para o looping e o continue pula o item da condição.

While

Possui o mesmo conceito de looping do for mas é mais utilizado quando não sabemos quando será feito a parada da condição. Iniciamos a variável fora do while e incrementamos a mesma dentro do looping.

For of

Atribui cada um dos caracteres ou dos elementos de uma array a uma nova variável para ser utilizada no looping individualmente sem afetar o original.

For in

Cria um looping em cima de um objeto, similar ao anterior e podemos usar a variável criada para acessar as propriedades do objeto dinâmicamente.

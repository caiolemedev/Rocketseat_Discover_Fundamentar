JS - Tipos de dados

Introdução

Temos os elementos da linguagem e suas combinações (gramática) e o conjunto de termos e expressões (vocabulário). Precisamos entender o significado e como escrever primeiro. Apresentar os principais tipos de dados.

String

Cadeia de caracteres que formam textos, podemos usar: aspas duplas (quando dentro do texto tem aspas simples), aspas simples (quando dentro do texto tem aspas duplas) e o template literals (aceita os dois tipos de aspas dentro delas, permite escrita em multiplas linhas e aceita expressões dentro da string - interpolação).

Number

Identificadores de números, podem ser inteiros, reais/float (com ponto), NaN (not a number) e Infinity (infinito, usado para verificação).

Boolean

Variável que só tem 2 valores: true (verdadeiro) e false (falso).

Undefined vs null

Undefined é um dado indefinido, reservado pelo JS para coisas que não existe. Null é um objeto que não possui nada dentro e é diferente de indefinido.

Object

É um dado estrutural, que cria uma estrutura e possui propriedades (atributos) e funcionalidades (métodos). Deve ser criado com colchetes e dentro suas características com cada valor.

Array

São os vetores, é uma lista que agrupa os dados. Aceita diferentes tipos de dados no mesmo vetor desde que separados por vírgula e entre chaves.

Conclusão

De acordo com o ECMAScript Standard (orgão que padroniza a linguagem) existem 9 tipos de dados divididos entre primitive (valores que não sofrem alterações), structural e structural primitive. Os primitivos são: String, Number, Boolean, undefined, Symbol, BigInt. Os estruturais são: Object (array, map, set, date...) e Function. Primitivo estrutural: null (deveria ter um valor mas não tem).

Praticamente todos os valores e manipulação de dados feitas no JS são via objeto, portanto temos muitas opções dentro dessa categoria.

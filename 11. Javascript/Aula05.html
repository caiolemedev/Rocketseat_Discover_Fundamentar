Praticando e avançando

Declaration assignment var

Declaração não atibui valor para a variável, podemos fazer uma atribuição depois. Podemos descobrir o tipo de dado dentro da variável pelo comando typeof.

Agrupando declarações

Podemos agrupar variáveis para declarar e depois atribuir valor para cada uma delas. No console log usamos vírgula para separa e imprimir todas de uma vez.

Conectando e interpolando variáveis

Podemos adicionar variáveis no texto escrito usando o sinal de mais (+) como no exemplo ou podemos usar o template literals (interpolação).

Objects

Revisão rápida sobre objeto, podemos usar o nome do objeto seguido de ponto e o nome da propriedade para acessar seu valor.

Arrays

Para acessar o valor de um item dentro do array usamos o nome do array seguido de chaves e dentro o número referente ao espaço ocupado pelo item, lembrando que a contagem começa no 0. Para acessar a propriedade lenght do array usar o conceito de objeto, com ponto. Padronizar os arrays para não ficar com o código confuso.

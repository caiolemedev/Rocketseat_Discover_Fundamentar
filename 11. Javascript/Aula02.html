JS - Primeiros Passos

Sintaxe

Maneira correta de escrever uma linguagem, no caso Javascript tem sua própria maneira de ser escrita para funcionar. O VS code ajuda a contornar alguns erros.

Maneiras de executar o JS

No próprio navegador da nossa máquina (ferramentas de desenvolvedor em console), via codepen.io que permite rodar o código e finalmente no VS code (dentro via tag script dos arquivos HTML ou fora via JS externo).

Adicionando arquivo JS

Via tag script dentro do html indicando a fonte (source).

Comentários

Inserindo comentários em linha com // ou em bloco com /* */.

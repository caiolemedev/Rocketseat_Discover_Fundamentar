Funções

Functions

Para começarmos uma função precisamos primeiramente declarar a mesma (function statement or declaration), cada função inicia um novo escopo. A função é guardada em memória, precisa ser executada/chamada/rodada (run, execute, call) posteriormente. Caso a função seja chamada sem ter sido criada o sistema apresentará um erro de referência.

Argumentos e parâmetros

A função pode ser salva dentro de uma variável sem um nome próprio (anônima), chamada function expression ou anonymous. É executada chamando a variável como nome da função. Podemos enviar parâmetros para a função, não é necessário declarar os argumentos que a função receberá pois isso é feito na chamada da função.

Retornando valores dentro da função

Podemos retornar o valor de uma função através do comando return e atribuindo isso a uma variável. Caso não seja colocado um delcarador antes da variável interna da função ela existirá fora da função (não devemos fazer isso para não causar confusão!!).

Outra maneira de entender funções

Explicando funções como se fosse um liquidificador.

Function Scope

Caso a função altere uma variável interna com o mesmo nome de uma externa, a mesma não será modificada caso seja passada como argumento, mas caso a função não tenha argumento a variável será alterada para todo o resto do código após a função ser executada.

Funtion Hoisting

Quando feita anônima com uma variável ela não sofre elevação, mas se feita da maneira tradicional sim e pode ser executada normalmente.

Arrow function

Jeito mais moderno de esrever funções via variáveis, usamos o formato de arrow (seta) para atribuir a função a uma constante.

Callback function

Funções que chamam outras funções externas que são recebidas através dos argumentos se tornando parâmetros para serem rodados dentro dela.

Funções construtoras

Usando a expressão new antes da atribuição de uma função a uma variável ela se torna uma construtora e cria um objeto dentro da variável. Com isso usando a palavra this dentro da função podemos acessar o objeto criado e adicionar propriedades, assim podemos criar diversos objetos com os mesmos padrões de propriedade. É recomendado usarmos a primeira letra maiúscula pois já existem funções construtoras padrões assim no JS (String, Number, Date).

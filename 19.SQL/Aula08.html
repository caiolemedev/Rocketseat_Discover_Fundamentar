Operadores Lógicos

AND

Usamos o comando AND quando desejamos juntar mais de uma condição no WHERE, os resultados apresentados deverão preencher ambos os critérios. Exemplo: SELECT * FROM aluno WHERE nome like 'j%' AND matricula = 1.

OR

Similar ao anterior mas traz resultados que cumpram um ou outro critério.

BETWEEN

Usamos a sintaxe como no exemplo: SELECT * FROM funcionarios WHERE id_funcionario BETWEEN 4 and 7. Se adicionarmos NOT antes do BETWEEN a query irá trazer os demais dados.

IN e NOT IN

Usamos como no exemplo: SELECT * FROM funcionarios WHERE id_departamento IN (1,2,5). Se adicionarmos o NOT o comando trará o oposto, os que não estiverem na seleção.

IS NULL e IS NOT NULL

Usamos como no exemplo: SELECT * FROM funcionarios WHERE id_departamento IS NULL. Se adicionarmos o NOT o comando trará o oposto, todos que não são NULL. O campo representado com NULL indica que o espaço em questão não foi preenchido.

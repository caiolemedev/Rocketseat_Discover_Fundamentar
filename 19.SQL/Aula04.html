Tipos de campos

Text

Precisamos informar o banco de dados qual o tipo de dados de cada coluna. O tipo texto será todo tipo de dado que possui letras ou caracteres especiais.

Number

O tipo de dado number é usado para informações que possuem somente numeros, sem espaços, traços ou outro caracter qualquer.

Datatime

Este campo guardará informações de data e hora numa configuração específica que o banco de dados entende como uma informação de data e hora e não texto ou número.

Primary Key

Será o identificador dos dados, é a informação que não pode se repetir. Informamos ao banco de dados que esta será a referência através do Primary Key. Normalmente usamos campos numérios. Exemplos id user: number primary key.

Nomes de campos e tabelas

Para escrevermos precisamos sempre começar com letras do alfabeto, não é permitido iniciar com caracter especial ou número. Alguns caracteres não são permitidos como o parenteses, sinal de mais e etc. Não podemos ter espaços no nome dos campos, usamos snakecase ou camelcase. Acentuação e cidilha também não são permitidos.

Conceitos

Tabelas

As informações são guardadas em forma de tabela, onde cada linha representa um conjunto de dados associados ao mesmo item. Todas possuem cabeçalho para indicar qual tipo de dados estão listados.

Campos

Os campos são os nomes indicados no cabeçalho, ou seja, os títulos das informações que guardamos na tabela.

Informações

As informações são os dados que são guardados em cada espaço da tabela.

Exemplo escola

Criando uma tabela básica de alunos e professores com informações simples. Identificamos os alunos por um número de matrícula, que será único para cada um, este estará associado a um conjunto de dados. De forma similar usamos id de professores para construir uma outra tabela. Os números de matrícula de alunos e id de professor podem ser iguais pois não terão nenhuma relação.

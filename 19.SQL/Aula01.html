O que é um banco de dados?

Repositório sistêmico de informações - guarda informações de forma organizada. Exemplo: lista de contatos do celular. Banco de dados veio substituir os arquivos físicos com o objetivo de agilizar a consulta e minimizar o espaço de armazenamento.

Serviços de banco de dados

Existem algumas empresas que prestam serviço para podermos guardar nossos banco de dados. Exemplo: SQL, MongoDB, MySQL, Oracle, etc.

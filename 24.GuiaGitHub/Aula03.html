Criando repositórios

Criando repositório online

Usando o próprio GitHub podemos criar um repositório e edita-lo online, por exemplo o README.md do perfil usando o nome da conta como repositório.

Conhecendo a página do repositório

Apresentando as features da página do repositório como brench, code, actions, pull requests, etc.

Atualizando o repositório

Atualizando o README.md no site do GitHub clicando no lapis. Fazemos as alterações necessárias como sites, contato, resumo, tecnologias, etc.

Copiando ideia de README pessoal

Podemos entrar na página de alguém, acessar os repositórios, acessar o README.md e clicar em raw. O arquivo .md aceita algumas coisas de html também.

Colaborando em outros repositórios

Podemos propor mudanças em repositórios de outras pessoas clicando em editar e adicionando um comentário. Essa alteração é enviada via pull request para o dono do repositório. Quando fazemos isso o GitHub também cria um fork e traz para o nosso usuário o repositório alterado.

Conhecendo o restante da página pessoal

Podemos colocar os repositórios favoritos na página principal. O GitHub também mostra os commits que fazemos no dashboard.

Estutura de um formulário

Form

A tag form vai definir o formulário e possui atributos básicos de action e method. O action recebe os dados, caso esteja vazio será carregado na mesma página. Método será GET (menos dados) ou POST (mais dados), caso esteja vazio será considerado GET. Não existe form dentro de form.

Fieldset legend

Fieldset é usado para agrupar elementos de um formulário e aceita parâmetros como disable (não envia os dados), form (liga o fieldset ao formulário), name (necessário para identificar o elemento) e legend (nome do agrupamento e é o primeiro parâmetro do formulário).

Label

Elemento label serve para associar ou identificar uma ou mais tags de entrada de dados, ajuda na acessibilidade e permite o clique para a entrada de dados. Para colocar o input fora do label precisamos usar for no label e o id no input (só funciona com elementos específicos).

Button

Usado para enviar ou resetar o formulário, vem com estilo padrão do navegador mas pode ser modificado. Usar o parâmetro submit para enviar e reset para limpar. Aceita o parâmetro autofocus e já é iniciado selecionado. Podemos adicionar disable para manter bloqueado durante o preenchimento via JS. Name e value no button servem para enviar um parâmetro ao submeter o formulário. E a tag form para linkar o botão com o formulário caso o botão esteja externo ao formulário.

Datalist

Fornece para o usuário uma lista de sugestão para um input específico, não são valores obrigatórios. Só mostra valores aceitos pelo input. Verificar compatibilidade no caniuse.com.

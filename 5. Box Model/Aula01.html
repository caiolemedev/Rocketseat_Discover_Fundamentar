Box Model

Introdução

Fundamental para fazer um layout para a web e facilita a aplicação do CSS. O box model é uma caixa retangular 2D que possui as seguintes características: width, height, content, border, padding and margin.

Box sizing

Responsável pelo tamanho base do objeto. O tamanho é aplicado ao conteúdo, portanto podemos ter uma caixa maior que sua width ou height definidos. Para definirmos o tamanho pelas bordas, usamos a propriedade box-sizing conforme no exemplo.

Display block e inline

Block: o elemento ocupa toda a linha colocando o próximo elemento abaixo, width e height são respeitados, padding, margin e border irão funcionar normalmente. Inline: um elemento ao lado do outro, não respeita dimensões, padding e margin. Podem ser alterados via propriedade display, como no exemplo.

Margin

São os espaços entre os elementos e aceitam valores tipo length e percentage. Shorthand aplicado no sentido do relógio: top, rigth, bottom e left. Por padrão o chrome coloca margem de 8px no body. Margin colapse é quando as margens dos elementos se juntam para dar o espaçamento total e só acontecem na vertical (block). Margin auto centraliza o elemento horizontalmente.

Padding

Preenchimento interno da caixa, entre a borda e o conteúdo. Shorthand e propriedades similares à margin. Pode causar diferença na largura de um elemento. Não aceita valor auto.

Border

Bordas da caixa, verificar a referência do mdn para mais detalhes. Possui propriedades de estilo, largura e cor. Normalmente usada em shorthand somente como border, se quiser selecionar uma borda específica adicionar a posição. Colocar bordas diferentes precisa colocar as específicas depois. Outline: similar a borda mas não altera o tamanho da caixa, pode ser diferente de retangular e é mais usado pelo user agent para acessibilidade.

Começando

Iniciando um repositório

Acessar o diretório do projeto no terminal, usar git init para criar o repositório vazio e nele se encontra apenas o arquivo oculto .git.

O git guarda o histórico do projeto

No diretório .git irá ficar guardado todas as alterações, não podemos alterar ou deletar a pasta.

O primeiro commit

Podemos gerar um arquivo txt e atualizar o diretório. Usamos primeiro o add para preparar os arquivos e depois o comando: git comit -m "menssagem do commit".

Git log

Usando o comando git log podemos verificar o histórico de alterações com o id do commit, data e hora, autor e menssagem. Podemos pesquisar por author ou grep com expressões regulares.

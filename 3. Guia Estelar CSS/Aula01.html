Aprendendo sobre CSS

Conhecendo o CSS

CSS significa cascading style sheet e tem a função de mudar o estilo do HTML.

Comentários no CSS

Comentários ajudam a navegar e entender o código. Não afetam o código. Começão com barra asterísco e teminam com asterísco barra

Anatomia do CSS

São utilizados os seguintes componentes: selector (faz referência a tag do HTML que terá as propriedades), declaration (é o conteúdo do seletor), properties (cada propriedade modificada via CSS) e property value (valor que cada propriedade terá para refletir no HTML).

Seletores

Seletor global: *
Seletor por elemento ou tipo: h1, h2, p, div
Seletor por ID: #box
Seletor por classe: .red
Seletor por atributo, pseudo-class, pseudo-element e outros

Box model

O HTML trabalha com modelo de caixas. Então é possível formatar borda, margem, padding, etc. Isso afeta as dimensões, posicionamento, espaçamento e etc.

Origem do CSS

Três formas de se adicionar CSS num HTML: inline, no próprio arquivo ou em um arquivo separado (folha de estilo). Neste parágrafo tem um exemplo de estilo inline.

A Cascata

Escolha do brownser de qual regra do CSS aplicar, será feita na sequência: inline > style interno > style externo. E a leitura é de cima para baixo, então caso seja alterado no mesmo nível o último será o aplicado, o mesmo se aplica para ordem de importação em caso de mais de um arquivo de estilo.

Regra Important

Não é considerada uma boa prática e não é recomendado o uso, somente quando está usando uma biblioteca externa que impede a aplicação do estilo desejado. Basta colocar !important na propriedade que você deseja sobrescrever todas as demais tags.

At rules

Aplicadas no inicio do CSS e é identificada pelo início no arroba @. As mais comuns são: import, media (usada para um min-width específico, normalmente dispositivos móveis), font-face e keyframes (animações).

Shorthand

Junção de propriedades de forma resumida. Não considera propridades anteriores e valores não especificados serão substituídos pelo padrão. Geralmente a ordem das propriedades não importa em alguns casos, mas precisamos ter cuidado.

Funções

Nome da função seguido de parênteses que recebe argumentos. Por exemplo: rgb, calc, etc. Usado como exemplo no título.

Dev Tools

Usando a ferramenta Dev Tools no navegador para verificar os parâmetros do CSS. Possível de editar para verificar os efeitos e tentar atingir o resultado desejado.

Cuidados com a escrita

Verificar os espaços quando utilizar shorthand. Formatar o código de forma organizada usando indentação. Caso as propriedades não sejam escritas da forma correta o CSS não funcionará.

Abertura

Por que esse curso?

Para acessar um servidor remoto sem interface gráfica. Portanto precisamos saber os comandos no terminar para podermos operar dentro dele.

O que é Command Line?

GUI (Graphical User Interface) são as janelas, botões, mouse, vídeos, etc.

CLI (Command Line Interface) operação do computador somente com textos quando não há necessidade de interação gráfica. O shell para interpretar e executar os comandos, ele conversa com o kernel e ele é o responsável por comandar as operações. Terminal > Shell > Kernel > Hardware. Bash - bourne-again shell (linux).

Sintaxe

Maneira como escrevemos comandos no terminal. Primeiro escrevemos o comando (programa que está executando, é obrigatório), seguido de opções (muda o comportamento, detalha como o programa deve funcionar, geralmente começa com traço e tem só uma letra, quanto tem dois traços não podemos juntar os comandos) e finalmente os argumentos (onde o comando vai operar, caminho do arquivo ou diretório).

O que são e como utilizar módulos?

O require chama os módulos do node, sejam eles nativos, instalados ou criados. Usamos como exemplo o path que é um módulo nativo do node. Assim podemos utilizar módulos externos em nossas aplicações do node.

Criando módulos

Para criarmos um módulo usamos a veriável global module.exports e atribuímos um valor, objeto, número, string a ele. Este valor será enviado quando executarmos a função require na aplicação principal. Exemplo usando os arquivos require e exports.

Pegando informações do processo

Capturamos informações do processo usamos a variável global process, que é um objeto com vários dados. O argv do process traz um array com os processos que estão sendo executados, desde o node até os argumentos colocados após a chamada do arquivo. Assim podemos capturar estes dados como entrada e usarmos dentro da nossa aplicação.

Passando flags como argumentos

As flags são informações no formato --nome que recebe um valor que vem logo na sequência, exemplo: --name "Caio Leme".

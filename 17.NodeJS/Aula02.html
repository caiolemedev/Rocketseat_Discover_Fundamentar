Configurando o ambiente

Instalando o NodeJS

Entrar em nodejs, baixar e instalar a versão LTS. Processo padrão de instalação.

REPL

Read-Eval-Print-Loop, poder escrever o JS num ambiente node direto no terminal. Para entrar é só digitar node no terminal. Temos a variável global no lugar do window e document. Alguns módulos importantes: os, path, fs, http, stream, etc...

VS Code

Para usar o node é recomendado fazer via VS Code pela integração e extensão da ferramenta. Instalar a extensão omni para a configuração de cor do VS Code ficar igual a da rocketseat.

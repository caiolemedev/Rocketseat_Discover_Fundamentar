BestMe App: Interagindo com o aplicativo

O que vamos aprender?

Criaremos um aplicativo para tirar a melhor versão de nós mesmos. Vamos aprender a nos comunicar com o processo enquanto ele ainda estiver rodando, além de assincronismo.

Apresentando as perguntas com process.stdout.write

Usando o process standout podemos imprimir strings na tela similar ao processo de console.log mas com mais controle do que está acontecendo. Criamos um array de perguntas e uma função para imprimir as perguntas na tela.

Recebendo dados de entrada e assincronismo com process.stdin.on

Usando a função acima podemos ter dados de entrada e criamos uma assincronia, mas a função permanece ligada aguardando uma entrada de dados para ser executada. Para fecharmos o processo usamos o process.exit e assim após o fluxo da função o processo é finalizado.

Lógica para apresentar perguntas e salvar as respostas

Para capturarmos as respostas criamos um array e usamos o método push para incrementa-lo. Também inserimos um looping onde a função de perguntas é executada enquanto o array de resposta for menor que o de perguntas e usa como input da função ask o tamanho do array de respostas, assim podemos incrementar o quanto for necessário. Antes de sair do programa inserimos uma visualização das respostas.

Finalizando e revisando

O programa roda até encontrar o exit, mas podemos adicionar um evento final que ouve o exit e exibe uma menssagem final do programa, nela colocamos um texto com todas as reflexões e respostas do usuário.
